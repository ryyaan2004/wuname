API

Content Negotiation

WuName will respond to the following 'Accept:' values with appropriate content

  • text/plain - Content will be returned as a plain string.
  • application/json - Content will be returned as a JSON object { message: 'message' }
  • text/html - Content will be returned as an HTML page with a twitter bootstrap hero unit, containing the message and the subtitle.
  • application/xml - Content will be returned as a XML document.

Operations

Path Description
/wuami/:from Will return content of the form ':wuname'
For e.g. /wuami/crashoverride will return 'Wicked Genius'
/enterthewu/:from Will return content of the form ':from from this day forward you will also be known as :wuname'
For e.g. /enterthewu/crashoverride will return 'crashoverride from this day forward you will also be known as Wicked Genius'
